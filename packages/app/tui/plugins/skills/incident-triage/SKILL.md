# Incident Triage

A capability-profiles demo skill. When loaded, triage in this order:

1. Assess blast radius and severity.
2. Mitigate first (stop the bleeding) before diagnosing.
3. Establish a timeline and find the root cause.
4. Write the follow-up items.
