Adds a privileged CMS procedure used only by `PilotSwarmManagementClient.restartSystemSession()`. It validates that the target row is an active system session, deletes transcript events, turn metrics, metric summary rows, and child outcome rows for that deterministic system-session id, then marks the session row deleted with a terminal restart state so `cms_create_session` can recreate a clean replacement under the same id.
