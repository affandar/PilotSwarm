Role Disclosure Model

The same platform objects appear across roles, but each persona gets a different default depth.

User

  • Agent catalog and session chat
  • Summaries, files, and quotas
  • Theme and GitHub Copilot key
  • Friendly activity, optional details

Power User

  • Agent Studio with test chat
  • Tool, skill, MCP, and access policy
  • Crawlers, fact graphs, retrieval tests
  • Advanced session diagnostics

Administrator

  • Fleet and worker health
  • Raw logs and failure analysis
  • User, quota, and model usage controls
  • Cluster actions with audit trail
